Bienvenidos a la API de proyectos AI

Aqui publicamos los proyectos relacionados con modelos de inteligencia artificial.

Estas son algunas de las tecnologias utilizadas en los proyectos, se ha utilizado Flask como Framework de Python para consumir los modelos.

  • argparse: Biblioteca de Python para procesar argumentos de la línea de comandos.
  • clip: Parte de la biblioteca CLIP de OpenAI, que permite el procesamiento de texto e imágenes.
  • k_diffusion: Biblioteca para la generación de imágenes a partir de texto usando el modelo V-Diffusion.
  • sounddevice: Interfaz para acceder a dispositivos de sonido en Python.
  • torch: Biblioteca para aprendizaje profundo y cómputo tensorial.
  • wavfile: Módulo para leer y escribir archivos de audio WAV en Python.
  • mtranslate: Biblioteca para traducción automática de texto.
  • gtts: Biblioteca para convertir texto en habla (Text-to-Speech) usando Google Text-to-Speech API.
  • openai-whisper: Modelo de síntesis de habla de OpenAI.
  • ffmpeg-python: Enlace Python para la biblioteca FFmpeg, utilizado para procesar y manipular archivos multimedia.
  • mediapipe: Biblioteca para análisis de datos de medios, incluyendo el seguimiento de manos y rostros.
  • pygame: Biblioteca para el desarrollo de juegos y aplicaciones multimedia en Python.
  • opencv-python: Biblioteca para visión por computadora y procesamiento de imágenes.
