Bienvenidos a la API de proyectos AI

Aqui publicamos los proyectos relacionados con modelos de inteligencia artificial.

Estas son algunas de las tecnologias utilizadas en los proyectos, se ha utilizado Flask como Framework de Python para consumir los modelos.

  • argparse: Módulo de Python para analizar argumentos de la línea de comandos.
  • clip: Librería para copiar texto al portapapeles desde el terminal.
  • k_diffusion: Herramienta externa para añadir ruido a las imagenes
  • sounddevice: Librería para acceder a dispositivos de sonido desde Python.
  • torch: PyTorch, una biblioteca de aprendizaje profundo.
  • wavfile: Módulo de SciPy para leer y escribir archivos WAV.
  • mtranslate: Paquete para traducción automática.
  • gtts: Google Text-to-Speech, una interfaz para la API de síntesis de voz de Google.
  • openai-whisper: Herramienta de OpenAI para convertir texto en audio.
  • ffmpeg-python: Una interfaz de Python para FFmpeg, una herramienta multimedia.
  • mediapipe: Biblioteca para soluciones de percepción en tiempo real.
  • pygame: Biblioteca para desarrollo de videojuegos en Python.
  • opencv-python: OpenCV, una biblioteca de visión por computadora.
  • flask-socketio: Extensión de Flask para agregar soporte de Socket.IO a una aplicación Flask.
